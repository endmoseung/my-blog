블로그에 코드를 올릴 때 하이라이트가 안 되면 좀 답답하잖아? 그래서 신경 썼어.

## 간단한 버튼 컴포넌트

```tsx
function Button({ label }: { label: string }) {
  return (
    <button className="rounded-full px-4 py-2">
      {label}
    </button>
  );
}
```

이렇게 코드블록이 깔끔하게 하이라이트돼. 읽기 훨씬 편하지?

## 인라인 코드도

문장 중간에 `const x = 1` 처럼 짧은 코드도 자연스럽게 들어가.
